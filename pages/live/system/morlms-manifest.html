MorLMS Manifest

This page is the root manifest for the Moribund Institute learning system. Human readers should use the Catalog. Offline LMS tools can use this page to discover subject manifests, project manifests, and future downloadable learning paths.

root@moribund:~$ morlms sync --source moribund-institute --mode decentralized

For human readers

Use the Catalog

The Catalog is the human-facing table of contents for school subjects, projects, posts, labels, and archives.

Use the Manifest

The manifest is for apps and offline tools. It tells software what courses and project indexes exist.

Human links

Catalog Human-facing directory of subjects and projects. Posts Browse recent Blogger posts. Categories Human-facing category and label guide. Archive Chronological Blogger archive. Wandering Wikis Project hub for wiki walking and lexicographical exploration. Wandering Wikis Blog Dedicated external Blogger for the Wandering Wikis project.

Manifest strategy

This root manifest should stay small. It should not contain every lesson from every subject. Instead, it points to smaller subject and project manifests. This lets the Moribund Institute grow in a decentralized way.

Example: an offline app can fetch this page first, then choose whether to sync only Wandering Wikis, only Language, only History, or the whole learning constellation.
