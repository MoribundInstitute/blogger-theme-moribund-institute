Shelves in 600

600 Technology The main shelf for applied science, tools, engineering, software, and practical systems. Applied Science Science put to use: tools, methods, systems, materials, processes, and practical outcomes. Engineering Design, construction, mechanisms, structures, systems, devices, and problem-solving machinery. Computers Hardware, operating systems, desktop tools, file systems, interfaces, and computing habits. Agriculture Farming, food systems, plants in use, cultivation, tools, and practical land work. Home Economics Household systems, food, clothing, maintenance, domestic craft, and daily-life technology. Manufacturing Making things, materials, production, tools, craft processes, and industrial methods.

Use this class when the post is mainly about making, building, repairing, programming, applying science, or using tools. If the post is mainly about pure theory or natural explanation, also consider 500 Science. If it is about art, games, music, or aesthetics, also consider 700 Arts.

Build Guides Step-by-step instructions for apps, websites, embeds, scripts, packages, and tools. Code Lessons Programming explanations, syntax drills, refactor notes, and small reproducible examples. Interactive Exercises Tap-to-build, flashcards, quizzes, reading checks, and embed-based technical lessons.
